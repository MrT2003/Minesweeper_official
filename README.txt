*Guildance Using*

STEP 1: Clone the " main " branch to your computer
STEP 2: Go to the " src " and choose the main class and then run

*Note*
 - Here I use Intelliji (Community )with JDK 21 and some external libraries. So to make sure that everything run well , please use IntelliJi and JKD 21. If you cannot run program , please follow these step:

Step 1: Go to the " File " in the menu bar and choose " Project Structure ".
Step 2: Choose the Modules in Project Settings and choose Dependencies.
Step 3: If your error comes from SDK version, please modify the Module SDK to SDK 21.
Step 4: Then choose Apply and Press OK. 
Step 5: Run main again.
